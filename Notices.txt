# Redistribution of LPSOLVE library
To make the installation of [iqcToolbox](https://github.com/iqcToolbox/iqcToolbox) easier, this repository provides compiled executables, libraries, and mex-files  of the [LPSOLVE library](http://lpsolve.sourceforge.net/) version 5.5.2.0.
This includes archives attainable directly from [sourceforge](https://sourceforge.net/projects/lpsolve/), detailed below:
 - [lp_solve_5.5.2.0_dev_osx32.tar.gz](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_dev_osx32.tar.gz/download)
 - [lp_solve_5.5.2.0_dev_win64.zip](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_dev_win64.zip/download)
 - [lp_solve_5.5.2.0_dev_ux64.tar.gz](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_dev_ux64.tar.gz/download)
 - [lp_solve_5.5.2.0_exe_osx32.tar.gz](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_exe_osx32.tar.gz/download)
 - [lp_solve_5.5.2.0_exe_win64.zip](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_exe_win64.zip/download)
 - [lp_solve_5.5.2.0_exe_ux64.tar.gz](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_exe_ux64.tar.gz/download)
 - [lp_solve_5.5.2.0_MATLAB_exe_osx32.tar.gz](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_MATLAB_exe_osx32.tar.gz/download)
 - [lp_solve_5.5.2.0_MATLAB_exe_win64.zip](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_MATLAB_exe_win64.zip/download)
 - [lp_solve_5.5.2.0_MATLAB_exe_ux64.tar.gz](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_exe_win64.zip/download)

The source files used for compilation can be found [here](https://sourceforge.net/projects/lpsolve/files/lpsolve/5.5.2.0/lp_solve_5.5.2.0_source.tar.gz/download).
